HOW TO USE HASKELL
1. Create program in vs code using the extension: example.hs
2. Create main = {-code here-}
3. Open GHCI by navigating to the directory containing the .hs file and typing GHCI in the directory listing
4. Compile your program by using the ":load"(:l) command: ":Load {-program name-}" 
5. type "main" to run the main of your program
6. Use ":reload"(:r) to update the file if editing while testing
7. End session by using ":quit"(:q)
